Configure your Slack channel
1. Create a Slack account. Use the link given below to be redirected to slack onboarding page.
https://slack.com/signin#/signin
2. Create a workspace to it. And also create a channel in which your bot will work
3. Create an app by going to the following link: "https://api.slack.com/apps"
4. In the dashboard, click create a new app and select from scratch
- Give app name (your bot name)
- Select your workshop in which the app would function
5. Go to Incoming Webhooks and activate it
- Click on "Add new Webhook to workspace"
- Select the channel in which you want your bot to work. If no channel is created,
go to Slack and create a new channel
- Click on Allow You will be redirected to the incoming webhooks webpage.Deactivte incoming Webhooks now.

paste it in Interaction Studio UI.
